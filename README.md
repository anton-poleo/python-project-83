# Page analyzer

Утилита для анализа веб страниц


## Зависимости

- python 3.12+

- uv 0.8.3+

## Установка

```shell
git clone git@github.com:anton-poleo/python-project-50.git
cd python-project-50
uv build
uv package-install
```

### Запуск
````
make start
````
